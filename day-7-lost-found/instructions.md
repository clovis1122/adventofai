# Day 7: Lost & Found

## Instructions
Create a goose recipe that act as a Lost & Found Data Detective. The recipe will receive a folder (see `./data`) with data containing lost & found items. It will:

- Identify duplicates (blue scarf = BLUE SCARF)
- Standardize locations (ice rink = ice skating area)
- Categorize items (clothing, electronics, keys, etc.)
- Flag urgent items (electronics, jewelry, IDs)
- Match potential lost + found pairs
- Build a web app using a single index.html with CSS and JavaScript.

The webapp will contain the following:
- Display summary statistics
- Highlight urgent items
- Show matched pairs
- Organize by categories
- Be visually appealing and functional
- Have a festive winter theme
